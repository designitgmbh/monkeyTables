Composer

You can install the monkeyTables PHP backend through Composer, a package manager for PHP. Once you have the composer command line tool installed, you can type $ composer.phar require "designitgmbh/monkey-tables":"dev-master" For any additional information just checkout the gitHub read me!

Bower Component

You can install monkeyTables through Bower, a package manager for web components. Once you have the bower command line tool installed, you can type $ bower install monkeytables. You might need to run bower cache clean first to get the latest versions because of a repo switcheroo.
